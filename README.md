## บทคัดย่อ
โครงงานเรื่อง Morse Code Monitor มีวัตถุประสงค์ในการออกแบบและพัฒนาอุปกรณ์ที่สามารถแปลรหัสมอร์ส (Morse Code) เป็นตัวอักษรภาษาอังกฤษได้ พร้อมทั้งเชื่อมต่อกับระบบเครือข่ายอินเทอร์เน็ตเพื่อแสดงผลแบบเรียลไทม์ โดยใช้อุปกรณ์หลักคือบอร์ด Arduino UNO R4 WiFi ซึ่งมีความสามารถในการประมวลผลและเชื่อมต่อ Wi-Fi ได้ในตัว

ระบบประกอบด้วย ปุ่มกด (Push Button) สำหรับส่งสัญญาณรหัสมอร์สในรูปแบบจุด (.) และขีด (-) และมีปุ่ม Enter และ Delete สำหรับการยืนยันและลบข้อมูล โดยแสดงผลบนจอ LCD 16x2 เพื่อแสดงตัวอักษรที่ถอดรหัสแล้ว บอร์ด Arduino ถูกโปรแกรมให้สามารถแยกแยะระยะเวลาการกดปุ่มเพื่อระบุรหัสแต่ละตัว และแปลงรหัสมอร์สเป็นตัวอักษรภาษาอังกฤษตามตารางมาตรฐาน พร้อมส่งข้อมูลผ่านเครือข่าย Wi-Fi เพื่อแสดงผลแบบเรียลไทม์

ระบบสามารถตรวจจับสัญญาณมอร์สจากการกดปุ่มได้อย่างถูกต้อง แปลงเป็นตัวอักษรภาษาอังกฤษได้ครบถ้วน และแสดงผลบนจอ LCD ได้อย่างแม่นยำ อีกทั้งยังสามารถเชื่อมต่อและสื่อสารกับอุปกรณ์ภายนอกผ่านระบบ Wi-Fi ได้ตามที่ออกแบบไว้ โครงงาน Morse Code Monitor จึงทำงานได้ตรงตามวัตถุประสงค์ และเป็นแนวทางที่เหมาะสมสำหรับการประยุกต์ใช้ระบบไมโครคอนโทรลเลอร์ในงานสื่อสารข้อมูลเบื้องต้น รวมถึงเทคโนโลยี Internet of Things (IoT)

## จุดประสงค์ของโปรเจกต์
1. เพื่อให้ผู้ใช้งานสามารถเรียนรู้และเข้าใจหลักการเข้ารหัส Morse Code ผ่านการลงมือปฏิบัติจริง
2. เพื่อใช้ Morse Code เป็นเครื่องมือในการสื่อสารในสถานการณ์ฉุกเฉิน เมื่อต้องการติดต่อโดยการเข้ารหัสสัญญาณ
3. เพื่อพัฒนาเครื่องมือช่วยฝึกฝนและทดสอบความเข้าใจการเข้ารหัส Morse Code ได้อย่างง่ายและมีประสิทธิภาพ
4. เพื่อเพิ่มความสนุกและความเข้าใจในการเรียนรู้ Morse Code ผ่านการโต้ตอบกับอุปกรณ์จริง เช่น สวิตช์และไฟ LED

## รายละเอียดของโปรเจกต์
การเข้ารหัส Morse Code เป็นระบบสื่อสารที่เป็นที่รู้จักกันอย่างแพร่หลาย แต่ยังมีหลายคนที่รู้จักเพียงชื่อแต่ไม่เข้าใจวิธีการใช้งานจริง โปรเจคนี้ถูกออกแบบมาเพื่อให้ผู้ใช้งานสามารถเรียนรู้และฝึกปฏิบัติการเข้ารหัส Morse Code ได้อย่างเข้าใจ โดยอุปกรณ์จะรับข้อมูลผ่านทาง

Input 2 ส่วน ได้แก่
- สวิตช์ (Switch): ใช้กำหนดสัญญาณจุด (.) หรือขีด (-)
- ข้อความ Morse Code: ผู้เล่นป้อนข้อความเข้ามาในรูปแบบรหัส Morse

จากนั้นระบบจะทำการตรวจสอบรหัส Morse ที่ผู้เล่นป้อนเข้ามา โดย
- หากคำตอบ ถูกต้อง จะแสดงข้อความ “Correct!” บนจอ LCD พร้อมแสดงไฟ LED สีเขียว
- หากคำตอบ ผิด จะแสดงข้อความ “Wrong!” พร้อมแสดงไฟ LED สีแดง

โปรเจคนี้สามารถประยุกต์ใช้ได้ทั้งในเชิงการศึกษา การฝึกฝนการเข้ารหัสข้อความ และการสื่อสารในกรณีฉุกเฉินที่ไม่สามารถใช้เสียงหรือข้อความได้ตามปกติ
